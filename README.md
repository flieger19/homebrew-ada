# Flieger19 Ada

## How do I install these formulae?

`brew install flieger19/ada/<formula>`

Or `brew tap flieger19/ada` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "flieger19/ada"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
